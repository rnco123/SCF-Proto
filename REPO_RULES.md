# Repository Access Rules

## Write-Enabled Repos (Can Push)
- **SCF-Proto** ✅
  - Full read/write access
  - Can commit and push to remote

## Read-Only Repos (No Push)
- **trade-origin-admin** ❌
  - Read-only access
  - No commits or pushes allowed
  
- **trade-origin-api** ❌
  - Read-only access
  - No commits or pushes allowed
  
- **trade-origin-web** ❌
  - Read-only access
  - No commits or pushes allowed

These rules ensure clear ownership and prevent accidental changes to dependency repositories.
